# Job Portal Frontend

Frontend application for the Job Portal System, built with React.js, Next.js, Typescript and modern UI technologies.

---

## Backend Repository

https://github.com/quoctrongdev-prog/Job-Portal-BE

---

## Features

- Job search and application interface
- User authentication (Login / Register / Forgot Password)
- Role-based UI (Jobseeker / Recruiter)
- Recruiter profile for managing companies, jobs and applicants
- Jobseeker profile management
- Integration with backend APIs

---

## AI Features

- Resume evaluation
- Career guidence  
(Powered by Gemini API)

---

## Tech Stack

- Next.js
- React.js
- TypeScript
- Tailwind CSS
- ShadcnUI
